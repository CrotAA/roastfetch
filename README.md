### 特性

- 100+ 条动态精准毒舌（内存爆表、硬盘塞满、DE 丑爆、用户名土味全中）
- 零运行时依赖（单二进制）
- 每次运行都不一样，永远新鲜的伤害
- 支持所有主流 Linux 发行版

### 一键安装（最快 5 秒）

```bash
curl -sL https://raw.githubusercontent.com/CrotAA/roastfetch/main/install.sh | bash
```

或者用 Cargo：

```bash
cargo install roastfetch
```

### 手动下载最新 Release（适合不想装 Rust 的）

去 https://github.com/CrotAA/roastfetch/releases/latest 下载 `roastfetch` 二进制  
`chmod +x roastfetch && sudo mv roastfetch /usr/local/bin/`

### 贡献

- 想加更狠的 roast？直接改 `src/roasts.rs`
- 想加新 ASCII art？扔 `arts/` 文件夹
- PR 越多越毒越好
